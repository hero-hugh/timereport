API, update Time Report

Prerequisites

  1. Access token from BOX, read more here
  2. Some knowledge about GraphQl

 

Example of fetch logic

const apiToken = "YOUR_ACCESS_TOKEN";
const url = "https://api.box.developersbay.se/api/v1/graphql";


const fetchGraphQL = async (query, variables = {}) => {
  try {
    const response = await fetch(url, {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        Authorization: `Bearer ${apiToken}`,
      },
      body: JSON.stringify({ query, variables }),
    });
    const data = await response.json();
    if (!response.ok || data.errors) {
      throw new Error(data.errors ? JSON.stringify(data.errors) : "GraphQL error");
    }
    return data;
  } catch (error) {
    console.error("GraphQL Request Failed:", error);
    return null;
  }
};

Retrieve time reports

// File: graphql-schema.js


export const getTimeReportsQuery = `query GetTimeReports($first: Int!, $page: Int, $orderBy: [GetTimeReportsOrderByClause], $filters: GetTimeReportFilters) {
  
  timeReports(
    first: $first
    page: $page
    orderBy: $orderBy
    filters: $filters
  ) {
    totalCount
    edges {
      node {
        id
        date
        usage {
          ... on User {
            id
            type
            firstName
            lastName
          }
        }
        totalHours
      }
    }
    pageInfo {
      currentPage
      hasPreviousPage
      hasNextPage
    }
  }
}
`;

 

  • first: Int, by default 20 - Number of time reports received, optional parameter for pagination
  • page: Int, by default 1 - Page for pagination
  • orderBy: Object[] - Used for sorting 
    • column: String - Allowed values:"DATE" | "CREATED_AT" | "HOURS" | "FIRST_NAME" 
    • order: String - Allowed values: "ASC" | "DESC"
  • filters: Object - Used for filtering 
    • value: Int - A full numeric representation of a year, at least 4 digits 
    • label: String - A full numeric representation of a year, at least 4 digits in string format
    • value: Int - Numeric representation of a month 
    • label: String - A full textual representation of a month, such as January or March
    • value: Int - Allowed values: 1 | 0
    • name: String - Allowed values: "Open" | "Closed"
    • year: Object 
    • month: Object
    • statuses: Object

 

Here is an example of a fetch code in JavaScript, using the GraphQl schema above:

import { getTimeReportsQuery } from "./graphql-schema.js";


const variables = {
  "filters": {
    "year": null,
    "month": null,
    "statuses": {
        "value": 1,
        "name": "Active"
    }
  },
  "first": 20,
  "orderBy": [
    {
      "column": "HOURS",
      "order": "DESC"
    }
  ]
};
console.log("Fetching Data...");
const data = await fetchGraphQL(getTimeReportsQuery, variables);
console.log("Response data:", data);

Get single time report

Here is the GraphQl schema to get single timereport, you need only one required parameter - id - Int, ID of time report

 

// File: graphql-schema.js


export const getSingleTimeReport = `query GetTimeReport($id: ID) {
  timeReport(id: $id) {
    id
    date
    usage {
        ... on User {
        id
        firstName
        lastName
        }
    }
    totalHours
    timeReportEntries {
        id
        timeReportId
        usageFee
        type
        date
        hours
        comment
    }
  }
}`;

Update time report

A TimeReport object can receive the following fields:

    • id: Int [required]
    • supplierInvoiceNumber: String - Custom invoice number
    • timeReportEntries: Array- array of TimeReportEntry objects

A TimeReportEntry object can receive following fields:

    • id: Int | String (if you want to create a new entry with type "different_rate" you need put "new-<index>" into ID)
    • timeReportId: Int | String (ID of the time report, a required field when editing grouped time reports)
    • usageFee: Float - Only for type "different_rate"
    • type: String - Allowed values: "common" | "unspecified" | "different_rate"
    • comment: String - You comment for entry
    • hours: String - Hours in format "HH:MM". For common type max - "23:59", for other - "999:59"

Here is the GraphQl schema to update time report:

 

// File: graphql-schema.js


export const updateTimeReportMutation = `mutation UpdateTimeReport($id: ID!, $input: UpdateTimeReportInput!) {
  updateTimeReport(id: $id, input: $input) {
    id
    date
    usageFee
    usage {
        ... on User {
        id
        firstName
        lastName
        }
    }
    totalHours
    timeReportEntries {
        id
        timeReportId
        usageFee
        type
        date
        hours
        comment
    }
  }
}`;
