## Codebase Patterns
- BOX API uses GraphQL at `https://api.box.developersbay.se/api/v1/graphql` — client in `apps/api/src/lib/box-client.ts`
- The project uses `turbo` for build orchestration, `biome` for linting, `vitest` for testing
- Run API-only typecheck with `cd apps/api && npx tsc --noEmit` (root tsc will hit web JSX config issues)
- Run lint with `npm run lint`, tests with `npm run test`
- `fetchBoxGraphQL` uses `Record<string, unknown>` for variables — no compile-time type safety on query variables
- Tests mock `fetch` globally via `vi.stubGlobal('fetch', fetchMock)` in box-client tests
